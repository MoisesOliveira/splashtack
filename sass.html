Sass es un preprocesador de CSS.

Las hojas de estilo son cada vez más grandes, más complejas y más difíciles de mantener. Aquí es donde puede ayudar un preprocesador. Te permite usar funciones que no existen en CSS, como variables, reglas anidadas, mixins, importaciones, herencia, funciones integradas y otras cosas.


Ejemplo:

/* Define las variables para los colores primarios */
$color_1: #a2b9bc;
$color_2: #b2ad7f;
$color_3: #878f99;
/* usa las variables */
.div1{
background-color: $color_1;
}
.div2{
background-color: $color_2;
}
.div3{
background-color: $color_3;
}

Un navegador no comprende el código de Sass. Por lo tanto, necesitarás un preprocesador de Sass para convertir el código en CSS. Este proceso se llama transpilación.

Transpilar es un término para tomar un código fuente escrito en un idioma y traducirlo a otro.

Instalación

Hay varias formas de instalar Sass en tu sistema. Una de ellas es con el uso de el gestor de paquetes npm. Para utilizar el gestor de paquetes npm debes instalar nodejs desde la página principal https://nodejs.org/es/

Una vez tengas instalado nodejs puedes utilizar el gestor de paquetes npm que viene integrado con nodejs, abre la línea de comandos y escribe el siguiente comando para instalar sass globalmente.


npm install -g sass

Preprocesamiento

CSS por sí solo puede ser divertido, pero a lo largo del tiempo el proyecto crece al igual que las hojas de estilo son cada vez más grandes, más complejas y más difíciles de manter.

Aquí es donde puede ayudar un preprocesador como Sass. Te permite usar funciones que aún no existen en CSS, como variables, anidamiento, mixins, herencia y otras ingeniosas ventajas que hacen que escribir css vuelva a ser divertido.

Una vez que empieces a jugar con Sass, tomará tu archivo preprocesado y lo guardará como un archivo css.

  1. Abre tu editor favorito.
  2. Ve a menú > archivo > nuevo archivo.
  3. Crea un archivo sass llamado estilos con extensión .scss
  4. Escribe código sass en tu archivo y luego conviértelo a css para que pueda ser interpretado por el navegador.

Para compilar Sass abre tu consola de comandos y escribe el comando:

sass estilos.sass estilos.css

Automáticamente se compilará tu archivo Sass y creará un archivo css con la compilación actual del archivo sass. Este proceso lo tienes que hacer cada vez que quieras compilar tu archivo sass a css, puedes agregar la bandera de vigilancia --watch seguido del comando anterior para observar la compilación todo el tiempo que esté corriendo el observador en la terminal.

sass estilos.sass estilos.css --watch

Como puedes observar la bandera de vigilancia --watch te evita tener que compilar sass cada vez que realices cambios, esta bandera se encarga de hacerlo automáticamente por ti. Para dejar de ejecutar la observación puedes presionar en el teclado de tu ordenador la tecla ctrl + c.

Variables

Con las variables puedes almacenar información que puedes reutilizar a lo largo del flujo del código.

Las variables de sass te permiten almacenar datos de cadenas, números, colores, booleanos, listas, y valores nulos.

Sass utiliza el símbolo de dólar $ seguido de un nombre para declarar una variable:

$nombreDeVariable = valor;

El siguiente ejemplo declara 4 variables llamadas fuente, color, tamanoDeFuente, ancho, puedes usarlas cuando quieras:


Ejemplo:

$fuente: Helvetica, sans-serif;
$color: red;
$tamanoDeFuente: 18px;
$ancho: 680px;

body {
font-family: $fuente;
font-size: $tamanoDeFuente;
color: $color;

#container {
width: $ancho;
}
